## Authentication

All IstanaPro API endpoints are authenticated using **Basic Auth** (username:password), as specified in the OpenAPI file.

```json
"security": [
  {
    "basicAuth": []
  }
]
```

- **Username:** `{{basic_user}}`
- **Password:** `{{basic_password}}`
